### 数组下标的总结

#### 前提

​	对于一个 n x n 的二维数组，首先默认按照下标从零开始，则索引变量 i 和 j 的范围为 0 ~ n - 1

#### 关系公式

​	对角线上点的索引变量关系为 i  ==  j ，斜对角线上点的索引变量关系为 i  +  j  ==  n - 1 

#### 应用举例

​	索引包含对角线在内的上半三角数组元素(包含对角线元素)时

第一重循环为：

```
for(int i = 0; i <= n / 2; i++)
```

第二重循环为：

```
for(int j = i; j <= n - 1 - i; j++)
```
